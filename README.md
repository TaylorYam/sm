# sm — S&P 500 個股動能策略週報

每日自動更新的 S&P 500 個股動能策略網頁週報。

## Live
👉 [https://tayloryam.github.io/sm/](https://tayloryam.github.io/sm/)

## 自動化
- 後端引擎（[QuantCore](https://github.com/TaylorYam/QuantCore)）每日計算 S&P 500 個股動能評分
- GitHub Action 在 00:36 將最新 `index.html` 推到此 repo
- GitHub Pages 自動發佈

## 配套
- [sr](https://github.com/TaylorYam/sr) — 同設計的產業 ETF 輪動週報
- [spy-stock-dashboard](https://github.com/TaylorYam/spy-stock-dashboard) — 互動式 Streamlit 版本
- [TaylorYam.github.io](https://tayloryam.github.io/) — 個人首頁
